# Client Management System
This project is to manage clients

## Requirements
python 2.7.18
pip 20.3.4
php 7.4.3

## Installation

### HTTP Server
Copy files(server.php, sse.php) to htdocs\joseph
```
xampp\htdocs\joseph
 | server.php
 | sse.php
```

### admin & client
```
pip install -r requirements

joseph\Scripts\activate
python client.py

joseph\Scripts\activate
python admin.py
```
